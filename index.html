Dolar Red

What is Dolar Red?

"Dolar Red" is a calculated value meant to represent a Canadianized version of the Dolar Blue. Instead of the unofficial (or "free") exchange rate between USD and ARS, it's an unofficial exchange rate between CAD and ARS. It's a simple calculation as follows:

Dolar Red = Dolar MEP / USD-CAD exchange rate

Note: Dolar MEP is used for the calculation since it is closest to the exchange rate used by credit card providers.
